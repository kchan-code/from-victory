Mock notes

  • Pattern mirrors apps/web/components/pregame/PracticeFlow.tsx → StatePickerScreen: role="radiogroup", role="radio"/aria-checked, 64px cards, gold-accent active, bottom-anchored CTA.
  • Frame 2 shows the keyboard focus-visible gold ring and the in-CTA loading state on submit.
  • Frame 5 (tennis) is a forward-compat illustration only — tennis is v2 and not in SUPPORTED_SPORTS. Do not build it for MVP.
  • Copy is sentence-case body / mono-gold eyebrows / display CONTINUE — no emoji, no shame language, audience-safe ("you"/"player", never "kid").
  • One gold element per view; cobalt reserved for progress/focus only (here, focus rings).
